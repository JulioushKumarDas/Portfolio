## Sections

1. **Hero** - Welcome message with call-to-action buttons
2. **About** - Brief introduction and stats
3. **Dashboard** - Quick skills overview
4. **Projects** - Featured projects with descriptions
5. **Skills** - Technical skills with progress bars
6. **Contact** - Contact form and information

## How to Customize

1. Edit the HTML to change text, add more projects, etc.
2. Update colors in the CSS `:root` variables
3. Add your social media links in the footer
4. Replace project information with your own work
