# Deployment Instructions for Portfolio

## Option 1: Deploy to Netlify (Recommended)

### Step 1: Create a GitHub Repository
1. Go to https://github.com/new
2. Create a new repository named `portfolio`
3. Clone the repository to your local machine
4. Copy all portfolio files into the cloned repository
5. Push to GitHub

```bash
git add .
git commit -m "Add portfolio website"
git push -u origin main
```

### Step 2: Connect to Netlify
1. Go to https://netlify.com
2. Click "Sign up" and choose "Sign up with GitHub"
3. Authorize Netlify to access your GitHub account
4. Click "New site from Git"
5. Select your portfolio repository
6. Click "Deploy site"

**Your portfolio will be live at: `https://your-portfolio-name.netlify.app`**

---

## Option 2: Deploy to Vercel (Free Alternative)

1. Go to https://vercel.com
2. Sign up with GitHub
3. Click "New Project"
4. Import your portfolio repository from GitHub
5. Click "Deploy"

**Your portfolio will be live at: `https://your-portfolio-name.vercel.app`**

---

## Option 3: Deploy to GitHub Pages (Free)

1. Go to your repository settings
2. Scroll to "Pages"
3. Select "Deploy from a branch"
4. Choose "main" branch
5. Save

**Your portfolio will be live at: `https://your-username.github.io/portfolio`**

---

## Option 4: Manual Deployment Using Netlify Drop

1. Go to https://app.netlify.com/drop
2. Drag and drop your portfolio folder
3. Netlify will instantly deploy it

**Your portfolio will be live at a random URL that you can customize**
